# CoinCollector

## About the Project

CoinCollector is a simple 3D game developed in Unity 6 as part of the **Keuzedeel K0788 – Basis programmeren van games** for the **Software Developer (MBO 4 BBL)** program at Aventus.

The goal of the game is to collect all six coins before the timer reaches zero.

## Features

- Player movement (WASD / Arrow Keys)
- Collectible rotating coins
- Score system
- Countdown timer
- Win screen
- Game Over screen
- Sound effects
- Arena walls
- Simple UI
- Git version control using GitHub

## Built With

- Unity 6 (6000.4.3f1)
- C#
- Git
- GitHub Desktop

## Controls

| Key        | Action        |
| ---------- | ------------- |
| W          | Move Forward  |
| A          | Move Left     |
| S          | Move Backward |
| D          | Move Right    |
| Arrow Keys | Move Player   |

## How to Play

1. Start the game.
2. Move around the arena.
3. Collect all six coins.
4. Avoid running out of time.
5. Collect all coins to win.
